# BART Game - Balloon Analogue Risk Task

A gamified version of the Balloon Analogue Risk Task (BART) for HR assessment purposes.

## Play the Game

The game is available at: https://erdilarsoy.github.io/bart-game/

## Description

This game measures how a person balances reward and risk under uncertainty. It's designed for HR use to assess decision-making, risk-taking behavior, and learning from feedback.

## Technology

- Phaser 3 (Game Framework)
- Tone.js (Audio)
- ES6 Modules

## Setup

1. Clone the repository
2. Open `index.html` in a modern browser that supports ES modules
3. Or deploy to GitHub Pages (already configured)

## Game Structure

- 3 tutorial trials
- 60 main trials (20 low-risk, 20 medium-risk, 20 high-risk balloons)
- Results are exported as CSV with HR metrics
